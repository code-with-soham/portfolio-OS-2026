# Keyboard Shortcuts & Global Commands

Portfolio OS implements a robust suite of global hotkeys to emulate a real desktop environment and improve power-user workflow. These are handled centrally via the `useKeyboardShortcuts.js` hook.

## System Shortcuts

- **Win + D**: Show / Hide Desktop (Minimizes or restores all open windows)
- **Win + E**: Open File Explorer
- **Win + R**: Open Run Dialog
- **Win + X**: Open Power User Menu
- **Win + S**: Open Global Search (Search apps, files, projects, skills)
- **Win + W**: Open Widgets Panel
- **Win + L**: Lock Screen
- **Win + Space**: Toggle Floating AI Assistant
- **Alt + Space**: Alternate trigger for Command Palette / Quick Search
- **Ctrl + Shift + P**: Open Command Palette

## App & Window Controls

- **Ctrl + T**: Open new tab (BrowserApp context)
- **Ctrl + W**: Close current tab / window
- **Ctrl + Tab**: Cycle through open tabs
- **Ctrl + S**: Save file (VSCode / Notepad context)

## Presentation & Demo Controls

These are specifically designed for the **Recruiter Experience** and presentation flows:

- **F9**: Start **Guided Presentation Mode** (Orchestrated sequence highlighting key features with floating overlays).
- **F10**: Start **Auto Demo Copilot** (A completely automated UI showcase with typing effects and a final Health Score evaluation).
- **F11**: **Emergency Exit** (Instantly kills any active presentation, resets overlays, and stops orchestration timers).
